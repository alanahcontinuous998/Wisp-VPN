## 🔍 Troubleshooting

Most connection problems happen because of local firewall settings. If you cannot connect:

- Check your internet connection.
- Turn off other VPN tools that might conflict with Wisp-VPN.
- Restart the application.
- Make sure your antivirus software does not block the app.

If you still face issues, look at the logs inside the settings menu. These logs contain information about why a connection attempt might fail.

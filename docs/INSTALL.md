# Установка

## Системные требования

- Windows 10 (версия 1809+) или Windows 11
- Права администратора (для установки сетевого драйвера туннеля)
- ~50 МБ свободного места

## Установка из релиза

1. Откройте [страницу релизов](https://github.com/PostmanReminisce/wisp-vpn/releases/latest)
2. Скачайте последний установщик (`WispVPN-Setup-x.y.z.exe` или `.msi`)
3. Запустите установщик от имени администратора
4. После установки запустите Wisp VPN из меню «Пуск»

> Пока в разделе Releases пусто — сборки появятся по мере готовности клиента. Следите за [CHANGELOG.md](../CHANGELOG.md).

## Проверка подлинности установщика

После публикации первого релиза здесь появятся контрольные суммы (SHA-256) для проверки скачанного файла — см. страницу конкретного релиза.

## Удаление

Стандартным способом через «Параметры → Приложения» Windows, как любую другую программу.

## Сборка из исходников

Инструкции по локальной сборке клиента будут добавлены в [`client/README.md`](../client/README.md) по мере разработки самого клиента.

## Проблемы с установкой?

Опишите проблему в [Issues](https://github.com/PostmanReminisce/wisp-vpn/issues/new/choose) с шаблоном «Bug report» — укажите версию Windows и шаги воспроизведения.
